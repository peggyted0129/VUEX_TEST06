# 蛋糕電商網站 —— Emily Shop  [Demo](https://peggyted0129.github.io/VUEX_TEST06/)  

> Vue CLI 4 電商
> 這是一份以 Vue.js 為框架的前端練習作品，無任何商業用途

![Emily Shop首頁](https://upload.cc/i1/2021/01/13/SdH3jp.png)

## 簡介

此作品為 Vue.js 前端練習，功能包含：

- 前台：
  - 電商展示（商品分類、單一商品介紹）
  - 商品收藏功能
  - 購物車
  - 優惠碼套用
  - 下單 / 結帳
- 後台：
  - 管理者登入
  - 商品管理（建立、更新、刪除）
  - 優惠碼管理（建立、更新、刪除）
  - 訂單列表
  
## Skills | 使用技術
